Update the existing SpacetimeDB backend module in TypeScript with an incremental migration.

TABLES
- legacy_item (public)
  - Preserve the existing table and fields
- item_v2 (public)
  - Fields:
    - id: u64 (primary key)
    - value: string
    - version: u32

REDUCERS
- Preserve seed
- migrate()
  - Be idempotent
  - Copy missing legacy_item rows into item_v2 with version 2
- dual_write(id: u64, value: string)
  - Write the same new item to both tables
  - Set version 2 in item_v2
