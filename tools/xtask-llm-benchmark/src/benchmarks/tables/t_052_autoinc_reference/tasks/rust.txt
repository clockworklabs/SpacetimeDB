Write a SpacetimeDB backend module in Rust that safely creates related rows with an auto-incremented parent ID.

Define public parent and child tables. Both have auto-incremented u64 primary keys named id. Parent also has name: String. Child has parent_id: u64 with a B-tree index named by_parent and name: String.

Define create_family(parent_name, child_names). Insert the parent, capture the row returned by insert, and use that returned id for every child. Never calculate or assume the next auto-increment value.
