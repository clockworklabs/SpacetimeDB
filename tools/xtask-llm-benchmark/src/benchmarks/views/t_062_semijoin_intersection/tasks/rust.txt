Write a SpacetimeDB backend module in Rust that defines a query-builder semijoin view.

TABLES
- member (public), struct Member
  - Fields:
    - id: u64 (primary key)
    - name: String
- eligibility (public), struct Eligibility
  - Fields:
    - id: u64 (primary key)
    - member_id: u64 (btree index)

REDUCERS
- seed()
  - Insert member 1 named "Ada" and member 2 named "Grace"
  - Insert eligibility id 10 for only member 1

VIEW
- eligible_member (public, query-builder)
  - Use a semijoin
  - Return members only when a matching eligibility row exists
