Write a C# SpacetimeDB module with public Customer, Purchase, and LineItem tables. Customer has Id: ulong primary key and Name: string. Purchase has Id: ulong primary key and indexed CustomerId: ulong. LineItem has Id: ulong primary key, indexed PurchaseId: ulong, Sku: string, and indexed Visible: bool. Add a Seed() reducer with one complete visible chain. Expose rows containing LineId: ulong, CustomerName: string, and Sku: string through a public OrderLineDetail view by joining all three tables through indexed lookups.
