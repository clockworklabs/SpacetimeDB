Write a SpacetimeDB backend module in C# that defines a procedural view using an index and additional filters.

TABLE
- Content (public, accessor Content)
  - Fields:
    - Id: ulong (primary key)
    - Category: string (btree index)
    - Active: bool
    - Score: int

REDUCERS
- Seed()
  - Insert rows that separately fail the Category, Active, and Score checks
  - Insert one passing row

VIEW
- FeaturedContent (procedural, public)
  - First use the Category index for "news"
  - Then return only Active rows with Score at least 10
