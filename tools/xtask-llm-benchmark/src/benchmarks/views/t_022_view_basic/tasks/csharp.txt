Write a SpacetimeDB backend module in C# that defines a public table and an anonymous view that returns all rows from the table.

TABLES
- Player
  - Struct: Player
  - Fields:
    - Id: ulong (primary key, AutoInc)
    - Name: string
    - Score: uint

VIEWS
- AllPlayers
  - anonymous view (uses AnonymousViewContext)
  - public
  - Returns: List<Player> (all rows from the Player table)
