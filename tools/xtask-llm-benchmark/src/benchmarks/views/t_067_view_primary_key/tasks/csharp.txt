Write a SpacetimeDB backend module in C# that defines a procedural view with a primary key.

TABLE
- SourceRow (public, accessor SourceRow)
  - Fields:
    - Id: ulong (primary key)
    - Value: string
    - Visible: bool (btree index)

VIEW
- SourceView (anonymous procedural, public)
  - Return visible SourceRow rows
  - Explicitly declare Id as the view primary key
