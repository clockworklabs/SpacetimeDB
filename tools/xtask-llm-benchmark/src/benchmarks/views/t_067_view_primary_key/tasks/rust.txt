Write a SpacetimeDB backend module in Rust that defines a procedural view with a primary key.

TABLE
- source_row (public), struct SourceRow
  - Fields:
    - id: u64 (primary key)
    - value: String
    - visible: bool (btree index)

VIEW
- source_view (anonymous procedural, public)
  - Return visible source_row rows
  - Explicitly declare id as the view primary key
