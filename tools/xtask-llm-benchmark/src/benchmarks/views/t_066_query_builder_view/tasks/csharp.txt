Write a SpacetimeDB backend module in C# that defines a query-builder filter view.

TABLE
- Ticket (public, accessor Ticket)
  - Fields:
    - Id: ulong (primary key)
    - Status: string (btree index)
    - Title: string

REDUCERS
- Seed()
  - Insert one open Ticket and one closed Ticket

VIEW
- OpenTicket (public, query-builder)
  - Return the query-builder filter for Status "open"
  - Do not materialize rows manually
