Write a C# SpacetimeDB module with a public JobResult table containing Id: ulong primary key and Status: string. Define a private PrivateJob scheduled table for reducer RunPrivateJob with ScheduledId: ulong auto-increment primary key, ScheduledAt: ScheduleAt, and ResultId: ulong in that order. Add EnqueuePrivate(Id) to write Status "queued" and schedule that Id one millisecond later. RunPrivateJob must be scheduler-only, not client-callable, and update the result to "complete".
