Write a Rust SpacetimeDB module with a public job_result table (struct JobResult) containing id: u64 primary key and status: String. Define a private private_job scheduled table (struct PrivateJob) for reducer run_private_job with scheduled_id: u64 auto-increment primary key, scheduled_at: ScheduleAt, and result_id: u64 in that order. Add enqueue_private(id) to write status "queued" and schedule that id one millisecond later. run_private_job must be scheduler-only, not client-callable, and update the result to "complete".
