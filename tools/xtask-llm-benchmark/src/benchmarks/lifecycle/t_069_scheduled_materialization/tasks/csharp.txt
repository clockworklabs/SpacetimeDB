Write a SpacetimeDB backend module in C# that refreshes materialized state through a scheduled reducer.

TABLES
- MaterializedState (public, accessor MaterializedState)
  - Fields:
    - Id: ulong (primary key)
    - Status: string
    - Version: ulong
    - RefreshedAt: Timestamp
- RefreshJob (private, scheduled for RefreshMaterialized, accessor RefreshJob)
  - Fields, in this order:
    - ScheduledId: ulong (primary key, auto-increment)
    - ScheduledAt: ScheduleAt
    - StateId: ulong

REDUCERS
- StartRefresh()
  - Write Id 1 with Status "pending" and Version 0
  - Schedule state 1 for one millisecond later
- RefreshMaterialized(job: RefreshJob) (scheduled reducer)
  - Update state 1 to Status "ready" and Version 1
  - Set RefreshedAt to the scheduled reducer timestamp
  - Complete without any further client call
