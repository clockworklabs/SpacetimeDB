Write a SpacetimeDB backend module in TypeScript that demonstrates client lifecycle events.

TABLE
- event
  - Fields:
    - id: bigint (u64, primary key, auto-increment)
    - kind: string

LIFECYCLE HOOKS (use spacetimedb.clientConnected / spacetimedb.clientDisconnected)
- onConnect: insert event with kind="connected"
- onDisconnect: insert event with kind="disconnected"
