Write a SpacetimeDB backend module in C# that defines one table and a reducer that performs insert, update, and delete in one call.

TABLE
- User
  - Struct: User
  - Fields:
    - Id: ulong (primary key, auto-increment)
    - Name: string
    - Age: int
    - Active: bool

REDUCERS
- Crud: perform these steps in order
  - insert (Name="Alice", Age=30, Active=true) — Id is auto-assigned
  - insert (Name="Bob",   Age=22, Active=false) — Id is auto-assigned
  - update (Id=1, Name="Alice2", Age=31, Active=false)
  - delete Id=2
