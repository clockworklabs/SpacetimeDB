Write a SpacetimeDB backend module in C# that defines one table with a multi-column B-Tree index and seeds rows.

TABLE
- Log
  - Struct: Log
  - Fields:
    - Id: ulong (primary key, auto-increment)
    - UserId: int
    - Day: int
    - Message: string
  - Indexes:
    - by_user_day: btree(UserId, Day)

REDUCERS
- Seed: insert exactly these rows into Log (IDs are auto-assigned)
  - (UserId=7, Day=1, Message="a")
  - (UserId=7, Day=2, Message="b")
  - (UserId=9, Day=1, Message="c")
