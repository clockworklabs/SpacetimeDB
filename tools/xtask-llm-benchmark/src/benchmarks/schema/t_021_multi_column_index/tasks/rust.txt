Write a SpacetimeDB backend module in Rust that defines one table with a multi-column B-Tree index and seeds rows.

TABLE
- log
  - Struct: Log
  - Fields:
    - id: u64 (primary key, auto_inc)
    - user_id: i32
    - day: i32
    - message: String
  - Indexes:
    - by_user_day: btree(user_id, day)

REDUCERS
- seed: insert exactly these rows into log (IDs are auto-assigned)
  - (user_id=7, day=1, message="a")
  - (user_id=7, day=2, message="b")
  - (user_id=9, day=1, message="c")
