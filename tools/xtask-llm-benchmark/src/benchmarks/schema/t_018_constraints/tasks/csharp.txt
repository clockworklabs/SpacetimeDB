Write a SpacetimeDB backend module in C# that defines one table and seeds two rows.

TABLE
- Account
  - Struct: Account
  - Fields:
    - Id: int (primary key)
    - Email: string (unique)
    - Name: string
  - Indexes:
    - by_name: btree(Name)

REDUCERS
- Seed: insert exactly these rows into Account
  - (Id=1, Email="a@example.com", Name="Alice")
  - (Id=2, Email="b@example.com", Name="Bob")
