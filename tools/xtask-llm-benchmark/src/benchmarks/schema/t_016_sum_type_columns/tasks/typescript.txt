Write a SpacetimeDB backend module in TypeScript that uses sum types in multiple table columns.

TYPES
- Rect
  - width: number (i32)
  - height: number (i32)
- Shape (sum type / enum)
  - circle: number (i32 radius)
  - rectangle: Rect

TABLE
- drawing
  - Fields:
    - id: number (u64, primary key, autoInc)
    - a: Shape
    - b: Shape

REDUCERS
- seed: insert one row with two Shape values (id is auto-assigned):
  - a: Shape.circle(10)
  - b: Shape.rectangle({ width: 4, height: 6 })
