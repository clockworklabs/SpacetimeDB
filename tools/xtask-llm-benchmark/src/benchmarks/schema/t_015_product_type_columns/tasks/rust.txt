Write a SpacetimeDB backend module in Rust that defines product types and uses them as columns in a table.

TYPES
- Struct: Address
  - Fields:
    - street: String
    - zip: i32
- Struct: Position
  - Fields:
    - x: i32
    - y: i32

TABLE
- profile
  - Struct: Profile
  - Fields:
    - id: u64 (primary key, auto_inc)
    - home: Address
    - work: Address
    - pos: Position

REDUCERS
- seed: insert exactly one row into profile (id is auto-assigned)
  - (home=Address{street:"1 Main", zip:11111}, work=Address{street:"2 Broad", zip:22222}, pos=Position{x:7, y:9})
