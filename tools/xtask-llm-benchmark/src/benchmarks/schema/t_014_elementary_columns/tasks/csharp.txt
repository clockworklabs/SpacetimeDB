Write a SpacetimeDB backend module in C# that defines one table and seeds one row.

TABLE
- Primitive
  - Struct: Primitive
  - Fields:
    - Id: ulong (primary key, auto-increment)
    - Count: int
    - Total: long
    - Price: float
    - Ratio: double
    - Active: bool
    - Name: string

REDUCERS
- Seed: insert exactly this row into Primitive (Id is auto-assigned)
  - (Count=2, Total=3000000000, Price=1.5, Ratio=2.25, Active=true, Name="Alice")
