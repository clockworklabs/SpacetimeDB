Write a C# SpacetimeDB module with a POST /webhook handler accepting a pipe-delimited body event_id|sequence|value. Define a private ProcessedEvent table with EventId: string primary key. Define a public WebhookState table with Key: string primary key, LastSequence: ulong, and Value: string. Store state under Key "account". A newly applied event must return status 200 and body "applied". Duplicate event IDs must return status 200 and body "duplicate" without applying twice. Events whose sequence is not newer must return status 200 and body "stale" without replacing state. Register POST /webhook through an HTTP router.
