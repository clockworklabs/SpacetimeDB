- [ ] See others' cursors
- [ ] Cursors show name + color
- [ ] Cursor shows tool icon
- [ ] **Smooth cursor movement** ⭐
- [ ] Fade on inactive

### 3. Shapes

- [ ] Rectangle, ellipse, line work
- [ ] Stroke/fill colors
- [ ] Shapes sync to others

### 4. Selection

- [ ] Can select elements
- [ ] Move/resize/delete
- [ ] **See others' selections** ⭐

- [ ] Multiple layers work

- [ ] **Auto-unlock on leave** ⭐

- [ ] User list on canvas
- [ ] **Status: active/idle/away** ⭐

- [ ] **Auto-away works** ⭐

- [ ] Chat panel works
