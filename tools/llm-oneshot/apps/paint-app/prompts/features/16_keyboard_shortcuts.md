### Keyboard Shortcuts

* V - Select tool
* B - Brush tool
* E - Eraser tool
* R - Rectangle tool
* O - Ellipse/Oval tool
* L - Line tool
* A - Arrow tool
* T - Text tool
* S - Sticky note tool
* Delete/Backspace - Delete selected
* Escape - Deselect / Cancel operation
* Shift (hold) - Constrain proportions
