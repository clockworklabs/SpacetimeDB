# Unity Quickstart

Get a SpacetimeDB Unity game running in under 5 minutes.

## Prerequisites

- [Unity 2021.3+](https://unity.com/download) installed
- [SpacetimeDB CLI](https://spacetimedb.com/install) installed

<InstallCardLink />

## Create your project

```bash
spacetime dev --template unity my-spacetime-game
```

This command:
1. Creates a new Unity project with SpacetimeDB integration
2. Creates a SpacetimeDB module (Rust or C#)
3. Starts the local SpacetimeDB server
4. Publishes your module
5. Generates C# bindings for Unity

## Open in Unity

1. Open Unity Hub
2. Click **Add** and select the `my-spacetime-game/client` folder
3. Open the project

## Project structure

```
my-spacetime-game/
├── spacetimedb/          # Your SpacetimeDB module
│   └── src/
│       └── lib.rs        # Server-side logic
├── client/               # Unity project
│   └── Assets/
│       ├── Scripts/
│       └── SpacetimeDB/
│           └── module_bindings/  # Auto-generated types
└── README.md
```

## Next steps

- Edit the module to add your game tables and reducers
- Use the generated bindings in your Unity scripts
- See the [Unity Tutorial](/docs/tutorials/unity) for a complete multiplayer game example
- Check out [Blackholio](https://github.com/clockworklabs/SpacetimeDB/tree/master/demo/Blackholio) for a full game demo
