# Rust Quickstart

Get a SpacetimeDB Rust app running in under 5 minutes.

## Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) installed
- [SpacetimeDB CLI](https://spacetimedb.com/install) installed

<InstallCardLink />

## Create your project

```bash
spacetime dev --template basic-rust my-spacetime-app
```

This command:
1. Creates a new project with a Rust SpacetimeDB module
2. Starts the local SpacetimeDB server
3. Compiles and publishes your module
4. Generates Rust client bindings

## Project structure

```
my-spacetime-app/
├── spacetimedb/          # Your SpacetimeDB module
│   ├── Cargo.toml
│   └── src/
│       └── lib.rs        # Server-side logic
├── client/               # Client application
│   ├── Cargo.toml
│   └── src/
│       ├── main.rs
│       └── module_bindings/  # Auto-generated types
└── README.md
```

## Test your module

Call a reducer from the CLI:

```bash
spacetime call --server local my-spacetime-app your_reducer "arg1"
```

Query your data:

```bash
spacetime sql --server local my-spacetime-app "SELECT * FROM your_table"
```

## Next steps

- Edit `spacetimedb/src/lib.rs` to add tables and reducers
- Build your client application using the generated bindings

- Read the [Rust SDK Reference](/sdks/rust) for detailed API docs
